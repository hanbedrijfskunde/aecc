Quick Start - Local Server

Choose one of these methods to run a local server:

Python (Built-in)

python3 -m http.server 8000

Then visit: http://localhost:8000

Node.js (npx)

npx http-server -p 8000

Then visit: http://localhost:8000

VS Code Extension
